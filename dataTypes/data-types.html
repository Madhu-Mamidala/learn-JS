Data Types in JS

Number
String
Boolean
Undefined

Number

Floating Point , Numbers

Ex : let age = 23;
used for Decimals and Integer
String

Sequence of Characters

Ex : let firstName = "Madhu";
used for Text
Boolean

Logical type that can only be True or False

Ex : let fullAge = true;
used for Taking Decisions
Undefined

Value taken by a variable that is not yet defined

Ex : let children = undefined;
used for empty values
Null

Also means Null and Empty Value

Ex : let age = null
used for empty values
Symbol

Values that is Unique and Can not be changed.

Ex : const sym1 = Symbol();
used for unique property keys to an object that won't collide with keys any other code might add to the object
BigInt

Values that is Unique and Can not be changed.

Ex : const bigInt = BigInt(1234567890123456789012345);
used to enable JavaScript programmers to represent integer values larger than the range supported by the Number data type.
